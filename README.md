# Simple Orders App

A simple fullstack application for managing products and orders.

## Features
- Login & Logout
- Product CRUD
- Create Order with stock validation
- Order History
- Backend API with Express
- Frontend with React (Vite)
- SQLite Database
- Docker & Docker Compose

## Tech Stack
- Backend: Node.js, Express, SQLite
- Frontend: React, Vite
- Database: SQLite
- Container: Docker

## How to Run

```bash
docker-compose up --build
